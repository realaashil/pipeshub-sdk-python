# ConversationStatus

Current status of the conversation:
<ul>
<li><code>INPROGRESS</code> - AI is processing</li>
<li><code>COMPLETED</code> - Response ready</li>
<li><code>FAILED</code> - Error occurred</li>
</ul>



## Values

| Name         | Value        |
| ------------ | ------------ |
| `INPROGRESS` | INPROGRESS   |
| `COMPLETED`  | COMPLETED    |
| `FAILED`     | FAILED       |
