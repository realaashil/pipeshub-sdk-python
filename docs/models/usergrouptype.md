# UserGroupType

Group type:
- admin: System admin group (cannot be modified)
- standard: Default user group
- everyone: All users group (cannot be modified)
- custom: User-created custom group



## Values

| Name       | Value      |
| ---------- | ---------- |
| `ADMIN`    | admin      |
| `STANDARD` | standard   |
| `EVERYONE` | everyone   |
| `CUSTOM`   | custom     |
